# Browser tools

These tools follow the minimal no-MCP pattern and use `puppeteer-core` when available.

Commands:
- `node .m-spec/tools/browser/start.mjs`
- `node .m-spec/tools/browser/nav.mjs <url>`
- `node .m-spec/tools/browser/eval.mjs '<expression>'`
- `node .m-spec/tools/browser/screenshot.mjs [output.png]`
- `node .m-spec/tools/browser/pick.mjs`
